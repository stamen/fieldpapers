Making Canned Atlases

It’s easy to create “canned” Field Papers atlases by providing extra details up-front that are used as default values throughout the Make process.

Each setting below can be posted via plain HTML form to fieldpapers.org; all are optional.

atlas_title
atlas_text
atlas_location
Latitude, longitude, and optional zoom. Use getlatlon.com to make new ones.

Examples:
37.8045, -122.2712 15
37.8045 -122.2712, 15
37.8045,-122.2712
37.8045 -122.2712
atlas_provider
URL template.

Examples:
http://tile.stamen.com/toner-lite/{Z}/{X}/{Y}.png (Black & White)
http://tile.stamen.com/boner/{Z}/{X}/{Y}.jpg (Satellite + Labels)
http://tile.openstreetmap.org/{Z}/{X}/{Y}.png (OpenStreetMap)
http://tile.stamen.com/bing-lite/{Z}/{X}/{Y}.jpg (Satellite Only)
