Making Atlases From GeoJSON

It’s possible to generate atlases from raw GeoJSON files. This can be done in one of two ways. A GeoJSON file can posted as an attached file to fieldpapers.org, named “geojson_file”:

geojson_file

Alternatively, raw GeoJSON content can be posted to fieldpapers.org as a form value, named “geojson_data”:

geojson_data
